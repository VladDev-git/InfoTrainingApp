Archer Pull Up

 

Technique Summary:

- Pull to side with one arm

- Other arm assists as less as possible

- Scapula depressed end retracted

- Full range of motion (ROM)

 

Description:

Execution:
Start in a dead hang position. Pull yourself up to one side. Focus on pulling with the arm closer to you. The other arm just supports as less as needed. At the top this arm should be straight. Slowly lower yourself to dead hang. For the next repetition pull yourself to the other side.
